# 🗺️ Project Map Standard

All updates to `docs/PROJECT_MAP.md` must follow this standard.

## 📅 When to Update
- **Structural Change:** Creating/Moving top-level folders.
- **Data Flow Change:** Modifying how characters move from `tooling/` to `assets/` to `lib/`.

## 📝 Rules
- Keep the ASCII tree (if present) or the bulleted list accurate to the physical disk state.
- Ensure the "Data Flow" section correctly describes the current pipeline.
