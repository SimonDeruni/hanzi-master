# ⚡ Commands Cheatsheet Standard

All updates to `docs/COMMANDS_CHEATSHEET.md` must follow this standard.

## 📅 When to Update
- **New Script:** Adding a file to `tooling/`.
- **New Build Step:** Adding flavor-specific build commands or release steps.

## 📝 Rules
- Commands must be presented in copy-pasteable blocks.
- Categorize by: Build, Test, Data Pipeline, Release.
