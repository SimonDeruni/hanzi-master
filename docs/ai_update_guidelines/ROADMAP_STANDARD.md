# 🗺️ Roadmap Update Standard

All updates to `docs/ROADMAP.MD` must follow this standard.

## ✅ Checkbox Rule
- When a task is 100% complete, change `[ ]` to `[x]`.
- Partial completion is NOT allowed. Keep it `[ ]` until finished.

## 🚀 Phase Transitions
- If a full Phase (e.g., Phase 8) is completed, update its status header:
  `## Phase X: Title (✅ COMPLETE)`
- Add a one-line summary of the Phase completion date.

## 🏗️ New Tasks
- Any new planned feature must be added to the appropriate Phase or a new "Future" phase at the bottom.
