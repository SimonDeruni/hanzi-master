# 🖌️ UI/UX Standards Standard

All updates to `docs/UI_UX_STANDARDS.md` must follow this standard.

## 📅 When to Update
- **Branding Change:** New colors, fonts, or paper textures.
- **Haptic Logic:** Changing vibration patterns for success/error.
- **Animation Rules:** New global curves or durations.

## 📝 Rules
- Use HEX codes for colors.
- Use Flutter-specific terms for curves (e.g., `Curves.easeInOutQuart`).
