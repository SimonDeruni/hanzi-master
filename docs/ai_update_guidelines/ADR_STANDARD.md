# 🏗️ Architectural Decision (ADR) Standard

All updates to `docs/ARCHITECTURAL_DECISIONS.md` must follow this standard.

## 📅 When to Update
- **New Library:** Adding a package to `pubspec.yaml`.
- **Core Logic Change:** Switching algorithms (e.g., Hausdorff to something else).
- **Architecture Shift:** Moving logic to Isolates, changing folder patterns.

## 📝 Entry Template
```markdown
## [ID]. [Title]
- **Decision:** [What was chosen?]
- **Rationale:** [Why is this best for a solo-founder/AI project?]
- **Rule:** [A mandatory code-level constraint resulting from this choice.]
```
