# 🚀 Hanzi Master - Current & Future Functionalities

## 💸 Monetization & Premium Access (CURRENT)
- **The Scholar's Edition (One-Time Unlock):** A Freemium model offering the core HSK 1 course for free, with advanced tools locked behind a single purchase.
- **RevenueCat Engine:** Secure, cross-platform in-app purchase validation handling both Google Play and Apple App Store transactions without a custom backend.
- **Paywall Interface:** An elegant, native bottom-sheet modal explaining premium benefits, complete with secure purchase and restore flows.

## 🗺️ Galactic Map & Progression (CURRENT)
- **The Living Scroll (Constellation Map):** A celestial journey through the HSK 1 curriculum.
    - **Solar System Topology:** Units are visualized as galaxies. A central **Sun** (Radical/Root node) anchors the unit, while related characters orbit as **Planets**.
    - **"Smart Spiral" Curriculum:** Enforced pedagogical 6-unit progression (Origin, Elements, Humanity, Village, Journey, City).
    - **Roman Numeral Indexing:** Large radical families are split into multiple galaxies (e.g., "Water II") for visual stability.
    - **Thematic Constellations:** Independent characters are grouped into star clusters (`✨`), ensuring every word has a home.
- **Scroll of Origin (Onboarding):** A persistent, interactive golden scroll node that initiates users into Hanzi mechanics using real data.
- **Visual Mastery States:**
    - **Active:** Standard thematic colors.
    - **Mastered (Sun):** Transforms into a **Golden Amber** jewel.
    - **Completed Galaxy:** Sun transforms into a **Jade Green** jewel when the entire cluster is mastered.
    - **Review Due:** Nodes emit a soft Red pulse when SRS review is required.

## 🎓 Educational Ecosystem (CURRENT)
- **Character Identity Cards:** Every node opens a "Blueprint" sheet showing Pinyin, Definition, and structural decomposition (A + B = C).
- **Radical Identity Cards:** Sun nodes provide a deep-dive into origin, mnemonic, pinyin, and a preview of its entire galaxy.
- **Radical Mastery Gauntlet:** Special 3-stage lessons for Sun nodes:
    - **Tracing:** Basic stroke mastery.
    - **Semantic Forging:** Multi-choice "meaning essence" selection to build a word.
    - **The Hunt:** High-speed recognition grid to identify radical "children."
- **Shared Mastery:** linked logic that updates all identical radicals across the map simultaneously.
- **Structured Lesson Flow (The 4-Pillar System):** Discovery -> Recognition -> Writing -> Reinforcement.

## 🎨 Drawing & Intelligence (CURRENT)
- **High-Quality Medians (HanziVG):** Fluid, hand-drawn centerline paths for natural matching.
- **Disappearing Ink:** Real-time cleanup of messy traces on success.
- **Geometric Stroke Matching:** High-precision comparison using the **Simplified Hausdorff** algorithm.
- **Spatial Logic Hardening (Centroid Check):** Positional validation ensuring strokes are in the correct region.
- **Radical Proxy Mapping:** "Rescues" missing strokes for isolated radicals (like `阝`) by borrowing from parent characters (like `院`).
- **Adaptive Mastery Corridors:** Success buffer zones that shrink as your streak increases.
- **Network Data Rescue:** Automatic fetching of missing strokes from AnimCJK CDN with Unicode variant fallback.
- **The Scholar's Progression (Gamification):**
    - **Ink Points (XP):** Experience gained for calligraphy accuracy.
    - **Scholar Ranks:** Culturally relevant progression ladder (Novice -> Artisan -> Master).
    - **Daily Quests:** Incentivized daily retention hooks.
    - **Ink Stone Widget:** Beautiful thematic dashboard for progress tracking.
